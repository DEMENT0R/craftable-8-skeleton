# CRAFTABLE

## Requirements
 * PHP 8.0+
 * Supported databases:
   * MySQL 5.7+
   * PostgreSQL 9.5+
 * npm 5.3+
 * node 8.4+

## Installation
```sh
composer install
php artisan craftable:install
npm install
npm run dev
```
